Issue: The app says "Typing..." but no text appears.

Cause: macOS blocked the keystrokes (Accessibility Permissions).

Fix:

  1. Open System Settings > Privacy & Security > Accessibility.
  2. Find "Human Typing Bot" (or "Electron" / "Terminal" if running from code).
  3. REMOVE them completely using the ‚ûñ button.
  4. Restart the app.
  5. Click "Allow" when prompted.

Or run this command in Terminal to force reset:

tccutil reset Accessibility com.typingbot.app
